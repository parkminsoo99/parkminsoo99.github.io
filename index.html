Logan

Logan은 미국에서 잔디 깎기 서비스(Mowing)를 제공하는 스타트업으로, 잔디 관리 장비 소유자(Mower)와 잔디 소유주(Homeowner)를 연결해주는 매칭 플랫폼을 운영하는 기업입니다. 저는 Logan의 기술팀에서 인턴으로 근무하며, Next.js, Hasura, GraphQL을 활용해 “https://hellologan.co”웹사이트의 Mower 프로필 기능을 개발했습니다. 또한, AWS Route 53을 이용해 PostgreSQL 데이터베이스 서버를 설정하였으며, 배포 단계에서는 Hasura 엔진을 위한 AWS EKS 클러스터 내 파드를 YAML 파일을 통해 선언적으로 배포하며 이 과정에서 애플리케이션 로드밸런서(ALB) 설정을 통해 트래픽 분산을 효율적으로 구현

K-Shield.Jr

K-Shield.Jr 3기 보안사고 분석대응 과정을 수료하며, 보안 이벤트 대응, 보안 로그 분석, 악성코드 분석, 침해사고 대응, 디지털 포렌식 등 다양한 분야의 교육을 이수하고 실습 경험을 쌓았습니다. 특히, 웹 해킹에 깊은 관심을 가지고, Kali Linux 환경에서 Ettercap을 활용하여 VMware에 다수의 가상 서버를 설치하고, ARP 스푸핑을 통해 웹사이트 로그인 과정에서 발생하는 이벤트를 트리거로 활용해 다른 사이트로 리디렉션되는 과정을 포착하여 해당 로그인 ID와 비밀번호를 탈취하는 실습을 진행했습니다. 또한, SQL Injection을 통해 데이터 조작 및 트래픽의 목적지(destination)를 공격자의 PC로 전송하는 공격을 실습을 경험
