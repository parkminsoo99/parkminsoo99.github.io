게임 친구 매칭 플랫폼(Diguru) - 프론트 게임 친구 매칭 플랫폼(Diguru) - 서버

게임 친구 매칭 플랫폼을 직접 설계하고 개발했습니다. 프론트엔드에서는 Next.js로 무한 스크롤 기반의 친구 추천 피드를 구현하고, React Query로 API 응답을 캐싱하며 Zustand를 활용해 전역 상태를 관리했습니다. 또한 SockJS와 STOMP를 적용해 실시간 채팅과 친구 요청 알림 기능을 제공 예정입니다. 백엔드는 NestJS와 Supabase OAuth를 기반으로 사용자 인증과 세션을 관리하고, TypeORM으로 다중 조인 쿼리를 최적화하며 예외 처리 구조를 계층화했습니다. 인프라 영역에서는 ESXi 환경에서 가상 서버를 구성하고 pfSense 방화벽과 Snort 침입 탐지 룰을 적용하여 보안성을 강화했습니다. 기획부터 개발, 보안 아키텍처 설계까지 전 과정을 직접 수행하며 웹 서비스 운영에 필요한 풀스택·인프라 역량을 모두 다졌습니다.

Next.js / NestJS / Supabase OAuth / PostgreSQL / ESXi / pfSense / Snort / SockJS
